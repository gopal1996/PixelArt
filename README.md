# Pixel Art Maker

These are the main sections of the WebApp.

## Deployment from master:
>   https://gopal1996.github.io/PixelArt/

## Features

* **Undo**
* **Redo**
* **EyeDropper**
* **Download as PNG**
* **Theme Changer**
* **Toggle Grid**
* **Reset**
* **Eraser**
* **Dynamic Grid**
* **Color Picker**

## Prerequisites

* **HTML 5**
* **CSS 3**
* **ES 6**
* **Javascript**
